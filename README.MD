## info
---


### 关于
---
info是一个放在系统托盘区域的一段简短信息，基于Electron实现

### 功能说明
---
目前只有最基本的功能
1. 多个信息管理
2. 每2秒刷新网页获取网页指定元素内容
3. 显示某个指定信息到托盘区域

正在开发中的功能
* 自定义刷新时间
* 同时显示多个信息（两个或以上）
* 信息显示多个样式可自由选择（例如现在是两行文字，上面是标题下面是信息，将会增加单行显示、只显示内容等样式选择）
